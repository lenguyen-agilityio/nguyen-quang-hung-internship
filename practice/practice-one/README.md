## GETTING STARTED

- Clone repo:

```
git clone https://github.com/lenguyen-agilityio/nguyen-quang-hung-internship.git
```

- Checkout branch:

```
git checkout feature/practice-one
```

- Install packages dependencies

```
npm i or yarn add
```

- Run Webpage

```
npm start or yarn start
```

- Open http://localhost:1234/ in browser
